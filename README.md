# 자산 매니저 v1.2

PWA 방식의 개인 통합 순자산 대시보드입니다.

## 포함 기능
- 총자산 / 총부채 / 순자산 대시보드
- 코인, 주식, 은행, 현금, 부동산, 자동차, 금, 기타 자산 관리
- 홍콩 모기지, 주담대, 차량할부, 신용대출 등 부채 관리
- KRW / USD / USDT / HKD / AUD 수동 환율 환산
- 자산군 비중 그래프
- 코인별 비중 그래프
- 계좌/거래소별 자산 그래프
- 순자산 추이 스냅샷
- 수정 / 삭제 / 필터
- GitHub 백업 / 복원
- JSON 파일 내보내기 / 가져오기
- 다크모드
- iPhone/iPad 홈 화면 설치용 PWA

## 사용 순서
1. GitHub 저장소에 파일 업로드
2. Public 저장소로 GitHub Pages 활성화
3. Safari에서 Pages 주소 접속
4. 공유 버튼 → 홈 화면에 추가

## 주의
소스코드가 Public이어도 자산 데이터는 기본적으로 기기 LocalStorage에 저장됩니다. GitHub 백업 버튼을 누를 때만 지정한 JSON 파일로 백업됩니다.
