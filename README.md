## 데이터 보호 방식

앱 데이터는 기존처럼 LocalStorage에 저장됩니다. v6.5부터는 데이터 변경 시 `assetManagerPWA_v6_backupHistory`에 최근 30개 버전 백업을 추가로 보관합니다.

대량 입력 전에 설정 탭에서 **현재 상태 버전백업**을 한 번 눌러 기준점을 만들어두는 것을 권장합니다.
