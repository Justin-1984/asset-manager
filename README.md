- Stable Base: v6.13.7
- UI Foundation: v6.16.x

- 시세/환율 갱신 결과를 긴 자산명 나열 방식에서 요약 방식으로 변경
- 기본 표시: 정상 여부, 시장 데이터 성공/전체, 환율 성공/전체
- 상세 버튼 추가
- 상세에서 캐시 유지, 실패, 제외 개수, 제외 자산 분류, 실패 항목, 소요시간 확인 가능
- 외화는 환율 갱신 영역으로 분리 표시

## 개발 원칙
- 데이터 구조 변경 최소화
- UI/UX 표시 방식 개선 중심
