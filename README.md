# AssetManager PWA v6.16.3 Home Dashboard Polish

기준 버전: v6.16.2 Market Summary Detail

## 변경 사항

- Home 화면의 시세/환율 상태 카드를 정리했습니다.
- 기본 화면에서는 긴 목록 대신 `정상 · 시장 n/n · 환율 n/n` 형식의 요약만 표시합니다.
- Home에서도 `상세` 버튼을 눌러 실패/제외/캐시/소요시간을 확인할 수 있게 했습니다.
- 기존 Settings의 상세 업데이트 로그 구조는 유지했습니다.
- 데이터 구조와 자산 계산 로직은 변경하지 않았습니다.

## 개발 방향

v6.16.x는 UI Foundation 단계입니다. 기능 추가보다 Home / Assets / Platforms의 가독성과 사용성을 우선 개선합니다.
