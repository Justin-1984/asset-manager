# 자산 매니저 PWA v6.3

## 변경 내용

- 업데이트 확인 버튼 추가
- 설치된 PWA에서 캐시 삭제 없이 최신 배포 파일 확인 가능
- Service Worker update / skipWaiting 처리 추가
- index.html, app.js, styles.css 강제 재조회 후 자동 새로고침
- v6.2 다크모드 버튼 수정 유지
- v6.2 입력 필드 이름표 유지
- v6.1 수정/삭제 기능 유지
- v6.0 분석 엔진 유지

## 포함 파일

- index.html
- app.js
- styles.css
- manifest.json
- sw.js
- README.md

## 사용 방법

GitHub Pages에 업로드 후 앱의 설정 탭에서 **업데이트 확인**을 누르면 최신 파일을 확인하고 자동으로 새로고침합니다.

최초 1회는 이 v6.3 파일을 GitHub Pages에 업로드해야 버튼이 생깁니다. 그 이후부터는 재설치 없이 업데이트 확인 버튼으로 갱신할 수 있습니다.
