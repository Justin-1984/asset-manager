# AssetManager PWA v6.16.12 Institution Select Icons

Stable cleanup 기반에 기관 선택형 입력을 전체 자산군으로 확장한 버전입니다.

## 변경 사항
- 코인: 거래소 선택 + 아이콘 자동 적용
- 주식/ETF: 증권사 선택 + 아이콘 자동 적용
- 은행/현금: 은행 선택 + 아이콘 자동 적용
- 보험: 보험사 선택 + 아이콘 자동 적용
- 목록에 없는 기관은 기타 직접입력으로 기존처럼 입력 가능
- 기존 한글/영문 기관명은 가능한 범위에서 자동 매칭

## 데이터 안전성
- localStorage 키 유지
- 기존 자산/거래/설정 스키마 변경 없음
- 기관명은 기존 country/account 필드에 계속 저장
