# Asset Manager v3.9

## 핵심 변경
- 거래소 잔고 API 자동조회는 보류
- 보유수량은 수동 입력
- 보유 코인의 현재가는 공개 시세 API로 자동조회
- 앱 실행 시 환율 자동조회 후 코인 시세 자동조회
- `전체 시세 갱신` 버튼 추가
- 시세 반영 후 평가액/수익률 자동 계산
