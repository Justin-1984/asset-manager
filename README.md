# 자산 매니저 v4.3

## 변경 사항
- 데이터 상태판 추가
  - 마지막 시세 갱신
  - 마지막 환율 갱신
  - 마지막 GitHub 백업
  - 마지막 GitHub 복원
- 최근 백업/복원 로그 표시
- 전체 갱신 버튼: 환율 + 시세 동시 갱신
- 미국주식/ETF 자동시세 추가
  - 예: SCHD, VOO, QQQ, JEPI, JEPQ
- 한국 ETF/국내주식 자동시세 1차 지원
  - 자산명에 6자리 종목코드 입력 권장
  - 예: TIGER 미국S&P500 → 360750
- 기존 코인 자동시세, 환율 자동조회, 월별 순자산 추이, 카드 결제예정액 유지
