# Asset Manager PWA v6.1 Edit Update

## 수정 내용

- 자산 카드에 `수정` 버튼 추가
- 부채 카드에 `수정` 버튼 추가
- 보험 카드에 `수정` 버튼 추가
- 수정 버튼을 누르면 기존 값이 입력창으로 자동 로드
- 저장 버튼이 `수정 완료`로 변경
- `수정 취소` 버튼 추가
- 삭제 전 확인창 추가
- v6.0 분석 엔진 유지
- 기존 v5.4/v6.0 데이터 마이그레이션 유지
- 캐시 버전 v6.1로 갱신

## 포함 파일

- index.html
- app.js
- styles.css
- manifest.json
- sw.js
- README.md

## 업데이트 방법

1. ZIP 파일의 6개 파일을 GitHub Pages 저장소에 덮어씁니다.
2. iPhone/iPad에서 설정 > 캐시삭제를 누릅니다.
3. 앱을 완전히 종료 후 다시 실행합니다.
4. 각 자산/부채/보험 카드에서 `수정` 버튼이 보이는지 확인합니다.
