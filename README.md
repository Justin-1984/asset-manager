# Asset Manager PWA v6.0 Fixed

## 긴급 수정 내용

- v6.0에서 기존 v5.4 데이터가 자동 인식되지 않아 예시 데이터가 표시될 수 있던 문제 수정
- 예시 데이터 자동 생성 제거
- 기존 LocalStorage 키를 넓게 탐색하는 마이그레이션 추가
- 설정 탭에 `기존 데이터 다시 찾기` 버튼 추가
- 설정 탭의 빈 검은 박스에 상태 메시지 기본 표시
- v6.0 분석 엔진 유지

## 포함 파일

- index.html
- app.js
- styles.css
- manifest.json
- sw.js
- README.md

## 사용 방법

1. 기존 v6.0 파일을 이 ZIP 파일 내용으로 교체합니다.
2. GitHub Pages에 업로드합니다.
3. iPhone에서 앱을 열고 설정 > 캐시삭제를 누릅니다.
4. 앱을 완전히 종료 후 다시 엽니다.
5. 데이터가 안 보이면 설정 > 기존 데이터 다시 찾기를 누릅니다.
6. 그래도 안 보이면 기존 백업 JSON으로 복원합니다.
