# AssetManager PWA v6.16.13 View Reset Fix

기관 선택형 입력 확장 버전에 자산 보기 검색 리셋과 버전 표시 오류 수정을 적용한 버전입니다.

## 변경 사항
- 브라우저 탭/문서 타이틀 버전 표시를 v6.16.13으로 정리
- 자산 보기 방식 변경 시 검색어 자동 초기화
- 자산 보기 요약 필터가 다른 보기 전환 후에도 남던 문제 수정
- 백업 파일명 v6.16.13 기준 정리
- 코인: 거래소 선택 + 아이콘 자동 적용
- 주식/ETF: 증권사 선택 + 아이콘 자동 적용
- 은행/현금: 은행 선택 + 아이콘 자동 적용
- 보험: 보험사 선택 + 아이콘 자동 적용
- 목록에 없는 기관은 기타 직접입력으로 기존처럼 입력 가능
- 기존 한글/영문 기관명은 가능한 범위에서 자동 매칭

## 데이터 안전성
- localStorage 키 유지
- 기존 자산/거래/설정 스키마 변경 없음
- 기관명은 기존 country/account 필드에 계속 저장
