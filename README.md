## 사용 방법

GitHub Pages에 업로드 후 앱의 설정 탭에서 **업데이트 확인**을 누르면 최신 파일을 확인하고 자동으로 새로고침합니다.

기존 데이터는 LocalStorage 구조를 유지하므로 삭제되지 않습니다. 업데이트 전 백업 다운로드를 한 번 눌러두는 것을 권장합니다.
