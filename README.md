# 자산 매니저 v1.5

통합 순자산 대시보드 PWA입니다.

## v1.5 변경사항
- 자산 검색 기능 추가
- 자산 정렬 기능 추가: 금액 높은순 / 낮은순 / 이름순 / 자산군순
- 수정 방식 개선: 수정 버튼을 눌러도 즉시 삭제되지 않고, 수정 저장 시 반영
- 삭제 전 확인창 추가
- 같은 날짜 스냅샷은 중복 추가 대신 업데이트
- 최근 스냅샷 목록 표시
- 순자산 추이 마지막 값 라벨 표시
- Service Worker 캐시 버전 v1.5로 갱신
