# 자산 매니저 v1.1

개인 통합 순자산 대시보드 PWA 초안입니다.

## 포함 기능
- 자산 입력: 코인, 주식, 은행, 현금, 기타
- 부채 입력: 모기지, 차량할부, 신용대출, 기타부채
- 총자산 / 총부채 / 순자산 자동 계산
- 자산군별 비중 도넛 그래프
- 로컬 저장
- GitHub 백업 / 복원
- 다크모드
- iPhone/iPad PWA 설치용 manifest + service worker

## 사용 방법
1. `index.html`을 브라우저에서 엽니다.
2. 자산과 부채를 입력합니다.
3. GitHub 백업을 쓰려면 별도 저장소와 Contents 권한 토큰을 입력합니다.
