# 자산 매니저 v6.16.1 Market Exclusion Detail

## 기준 파일
- v6.16.0 UI Foundation 정상 작동 확인본 기준

## 변경 사항
- 시세/환율 업데이트 결과에서 `제외된 자산명`을 함께 표시
- 제외 표시 형식: `제외: 자산명(기관)·자산종류`
- 실패 자산도 있을 경우 `실패: 자산명(기관)·자산종류`로 함께 표시
- 티커/코드 판별 우선순위 개선: symbol/ticker/code/name 순서
- 기존 데이터 구조, 자산 계산, 거래내역, 환율/시세 로직은 유지

## 목적
시세 갱신 결과의 `2개 제외`처럼 숫자만 보이던 부분을 실제 어떤 자산이 제외됐는지 바로 확인할 수 있게 개선했습니다.
