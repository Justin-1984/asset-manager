# 자산 매니저 PWA v6.0

## 이번 버전 핵심

v6.0은 분석 엔진 강화 버전입니다.

추가/강화 기능:

- 투자비중 점수
- 위험도 분석
- 코인 비중 분석
- 현금 비중 분석
- ETF/주식 비중 분석
- 단일 자산 집중도 분석
- 국가/통화/자산군 배분 계산 기반
- 리밸런싱 추천 문구 자동 생성
- 기존 LocalStorage 데이터 마이그레이션 구조 유지
- PWA 캐시 버전 갱신
- 백업/복원 기능 유지

## 포함 파일

- index.html
- app.js
- styles.css
- manifest.json
- sw.js
- README.md

## 배포 방법

GitHub Pages 저장소에 위 6개 파일을 업로드합니다.
기존 파일을 덮어쓴 뒤 브라우저에서 강제 새로고침하거나 앱 내 캐시삭제를 실행합니다.

## 주의

현재 v6.0은 분석 엔진 우선 버전입니다.
실시간 거래소 API 자동 수량 동기화는 아직 저장 구조만 유지하는 단계입니다.
