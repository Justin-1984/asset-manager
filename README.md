# 자산 매니저 v2.0

통합 순자산 대시보드 PWA입니다.

## v2.0 변경사항
- 환율 기준 화면 추가: USD / USDT / HKD / AUD
- USDT 환율을 USD와 분리 관리
- 외국 거래소 코인 입력 시 USDT 통화 자동 제안
- 자산 입력 중 원화 평가액 미리보기 추가
- 자산 목록에 적용 환율 표시
- 환율 마지막 저장 시간 표시
- Service Worker 캐시 버전 v2.0으로 갱신
