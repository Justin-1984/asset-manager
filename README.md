# AssetManager PWA v6.18.0 Institution Search

v6.16.13 안정판을 기준으로 기관 선택 UX를 확장한 버전입니다.

## 변경 사항
- 자산 종류에 따라 기관 목록 자동 변경
  - 코인: 거래소
  - 주식/ETF: 증권사
  - 은행/현금/예금/적금: 은행
  - 보험: 보험사
- 기관 검색 입력창 추가
- 검색 후 기관 선택 시 기존 country/account 필드에 표준 기관명 자동 저장
- 기타 직접입력 유지
- 기존 한글/영문 기관명 자동 매칭 유지
- 기관 아이콘 자동 적용 유지
- 브라우저 타이틀/버전 배지/캐시명/백업 파일명을 v6.18.0 기준으로 정리

## 데이터 안전성
- localStorage 키 유지
- 기존 자산/거래/설정 스키마 변경 없음
- 기관명은 기존 country/account 필드에 계속 저장


## v6.18.0 Cost Engine Cleanup
- 투자원금 입력을 평균 매입단가 기준으로 단순화했습니다.
- 내부 원금은 평균 매입단가 × 보유수량으로 자동 계산합니다.
- 외화 자산은 자산 통화 기준 원금 저장 후 KRW 환산합니다.
